## 🛠️ Tech Stack

### Core Frontend
- **React 19** - Modern UI components & hooks.
- **Vite** - Lightning-fast build tool.
- **Tailwind CSS 4** - Advanced utility-first styling.
- **DaisyUI 5** - Premium component library.
- **React Router 7** - Declarative routing.

### Animations & UX
- **Framer Motion** - Fluid physics-based animations.
- **GSAP** - Complex timeline-based interactions.
- **Lenis** - High-performance smooth scrolling.
- **Lucide React** - Sleek, lightweight iconography.

### Backend & Integrations
- **Firebase** - Real-time database & Auth.
- **Next.js** - Server-side rendering & API routes.
- **Stripe** - Secure payment gateway integration.
- **Redux Toolkit** - Scalable state management.

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/sanjidasefa/SanjidaSefaPortfolio.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
